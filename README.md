## Sudoku Solver
A simple sudoku solver which uses classical backtracking algorithm.It solves the sudoku game based on various difficulty level.


## Tech Stack
Frontend: 
HTML
CSS 
Javascript

Backend:
Flask

## Features
It can solve sudoku puzzle from image.
